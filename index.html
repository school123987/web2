How to get started with Brackets

-And also how to use Adobe Media Encoder

1. How to set up a webpage!

Start off by downloading the Brackets installer from here. Brackets is an open-source program with which you can code html files. Double-click the installer when it is finished downloading and follow the steps for installing Brackets. Once you've finished all these steps, you can begin to create your webpage.

The first thing you will wnat to do is copy and paste the HTML example code from here. After that, type in "Brackets" in the start menu and click it. Go to "File" at the top-right, and then click "New". Under the heading "Working Files", there should be a new file. Rename it to index.html. With this done, you can start coding in things you your website.

How to use Adobe Media Encoder to encode a video for your website.

Here are some helpful tips to using Adobe Media Encoder.

If you want to, you can reduce the byte size of a video by using some of Adobe Media Encoder's settings. You can do this by opening Adobe Media Encoder and clicking on the top left, at an option named "File". After clicking it, there should appear under it an option called "Add Source". Clicking on it. Find the video you want to reduce the size of and choose the "Open" button at the bottom right. Now inside the Adobe Media application, the video will be there under "Queue", at the top-right. Now right-click on the video and then left-click on export settings. At this point, a new window will open. Now, you can lower the size by lowering some of the numbers' sizes - for example, under video, you can reduce the resolution.
